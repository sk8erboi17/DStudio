Context size

          
How much the model keeps in mind per session. 64k is a good default; raise it for long agent or design work (DeepSeek V4 goes up to 1M).

          
        


        

          Good to know
          
Thinking & model — pick them per chat from the composer; in agent and design they restart the engine. Agent output stays structured via a small reversible patch built at launch (the ds4-agent source is never modified — Settings → Raw runs it stock). Network is localhost-only by default; switch LAN above to connect this machine to a DStudio host on the same network. Everything else lives in Settings (⚙).
